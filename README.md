# Tarkov RPG - 搜打撤游戏

一个受《逃离塔科夫》启发的2D俯视角RPG搜打撤游戏，使用 Godot 4.3 引擎开发。

## 游戏特色

- **搜**：探索地图，搜刮武器箱、医疗包、敌人尸体等获取装备
- **打**：与AI敌人（Scav/PMC）战斗，使用真实的弹道和后坐力系统
- **撤**：在规定时间内到达撤离点，带着战利品安全离开

## 核心系统

### 1. 玩家系统
- 身体部位伤害系统（头部、胸部、腹部、四肢独立血量）
- 体力系统（冲刺消耗、恢复）
- 技能系统（25种技能可升级）
- 等级和经验系统

### 2. 装备系统
- 装备槽位：头盔、面罩、耳机、护甲、背包、胸挂、主副武器
- 重量系统（影响移动速度）
- 护甲防护（减伤、耐久度）

### 3. 背包系统
- 格子化管理（不同物品占用不同大小）
- 仓库和战局背包分离
- 快速访问栏

### 4. 战利品系统
- 6种稀有度（普通到传说）
- 多种容器类型（武器箱、医疗包、保险箱等）
- 随机生成机制

### 5. 战斗系统
- 真实后坐力（垂直+水平）
- 子弹穿透和护甲计算
- 爆头倍率
- 换弹时间

### 6. 藏身处（基地）
- 设施升级（工作台、医疗站、比特币矿场等）
- 离线收益
- 治疗功能

## 控制方式

| 按键 | 功能 |
|------|------|
| W/A/S/D | 移动 |
| Shift | 冲刺 |
| C | 蹲伏 |
| 鼠标左键 | 射击 |
| 鼠标右键 | 瞄准 |
| R | 换弹 |
| E | 交互（搜刮、开门）|
| I / Tab | 打开背包 |

## 项目结构

```
tarkov-rpg/
├── scripts/           # GDScript脚本
│   ├── game_manager.gd       # 游戏管理器
│   ├── player_controller.gd  # 玩家控制器
│   ├── player_data.gd        # 玩家数据
│   ├── inventory_manager.gd  # 背包管理
│   ├── loot_system.gd        # 战利品系统
│   ├── enemy_ai.gd           # 敌人AI
│   ├── bullet.gd             # 子弹
│   ├── extraction_point.gd   # 撤离点
│   └── ...
├── scenes/            # 场景文件
│   ├── main_menu.tscn        # 主菜单
│   ├── hideout.tscn          # 藏身处
│   ├── player.tscn           # 玩家
│   ├── enemy.tscn            # 敌人
│   ├── bullet.tscn           # 子弹
│   ├── containers/           # 各种容器
│   └── maps/                 # 地图
├── resources/         # 资源文件
└── assets/           # 美术/音效资源
```
